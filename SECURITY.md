# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in packt, please report it privately via GitHub's security advisory feature:
https://github.com/BlackPool25/packt/security/advisories/new

Please do NOT file a public issue for security vulnerabilities.

## What to include
- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested fix (if known)

## Response
We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope
- Integrity: BLAKE3 hash verification correctness
- Storage: Data loss or corruption in pack files
- Supply chain: Compromised dependencies
- Memory safety: Unsafe code or undefined behavior
